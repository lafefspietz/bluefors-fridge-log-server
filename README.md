# [bluefors-fridge-log-server](https://github.com/lafefspietz/bluefors-fridge-log-server)

This is a local web server for handling data from a set of log files from a Bluefors dilution refrigerator. 

To install, first install the [local-web-server](https://github.com/lafefspietz/local-web-server) then replicate this repository into a subfolder of your folder at c:\local-web-server\. This could be the name of this repository or you could fork it with the name of your exact specific fridge.
